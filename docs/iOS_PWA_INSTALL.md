# рҹ“ұ Installing JiCompta on iPhone/iPad

Step-by-step guide to install JiCompta as a home screen app on iOS.

## вҡ пёҸ Important: Use Safari, Not Chrome

On iOS, **only Safari can install web apps to your home screen**. Chrome, Firefox, and other browsers cannot add PWAs to the home screen due to Apple's restrictions.

## рҹ“Ӣ Installation Steps

### Step 1: Open Safari

1. Open the **Safari** browser on your iPhone or iPad
2. Navigate to: `https://jicompta.web.app`
3. Wait for the page to load completely

### Step 2: Access Share Menu

1. Tap the **Share** button at the bottom of Safari
   - It looks like a square with an arrow pointing up: в–ЎвҶ‘
2. The share sheet will slide up from the bottom

### Step 3: Add to Home Screen

1. **Scroll down** in the share menu
2. Look for **"Add to Home Screen"** option
   - It has a plus icon: вҠ•
3. Tap **"Add to Home Screen"**

### Step 4: Customize and Confirm

1. You'll see a preview screen with:
   - The JiCompta icon (calculator design)
   - Default name: "JiCompta"
   - URL: https://jicompta.web.app
2. You can edit the name if you want (keep it short!)
3. Tap **"Add"** in the top right corner

### Step 5: Done! рҹҺү

The JiCompta icon will now appear on your home screen!

## рҹ”§ Troubleshooting

### "Add to Home Screen" option is missing

**Cause:** Safari restrictions or website issues

**Solutions:**
1. Make sure you're using **Safari** (not Chrome/Firefox)
2. Check you're on the website (not a bookmark)
3. Try force-closing Safari and reopening
4. Clear Safari cache: Settings вҶ’ Safari вҶ’ Clear History and Website Data
5. Update iOS to latest version

### Icon doesn't show correctly

**Solution:**
1. Delete the home screen icon
2. Clear Safari cache
3. Visit the site again in Safari
4. Re-add to home screen

### Opens in Safari instead of standalone

**Cause:** Not installed correctly

**Solution:**
1. Long-press the icon on home screen
2. Tap "Remove App"
3. Follow installation steps again
4. Make sure to tap "Add to Home Screen" (not "Add Bookmark")

### Status bar color is wrong

**Cause:** iOS theme settings

**Note:**
- Light mode: Blue status bar
- Dark mode: Darker blue
- This is normal and matches the app design

## рҹҡ« Why Not Chrome on iOS?

**Short answer:** Apple's App Store rules

**Explanation:**
- All iOS browsers must use Safari's WebKit engine
- "Add to Home Screen" is a Safari-exclusive feature
- Chrome on iOS is essentially Safari with Chrome UI
- This applies to all third-party browsers on iOS

**Exception:** On Android, Chrome CAN install PWAs!
